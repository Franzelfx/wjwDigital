# Installation und Verwendung der Software "FFE-AI-OCR"
## Installation
### Voraussetzungen
- Windows 10 (64bit) oder neuer
- Linux (64bit)
- Mac OS (64bit)

### Installation unter Windows
1. Installieren Sie Tessaract OCR für Windows:

   https://github.com/UB-Mannheim/tesseract/wiki

### Installation unter Linux
1. Installieren Sie Tessaract OCR für Linux:
   
   ```
   sudo apt install tesseract-ocr
   ```

### Installation unter Mac OS
1. Installieren Sie Tessaract OCR für Mac OS:
   
   ```
   brew install tesseract
   ```
